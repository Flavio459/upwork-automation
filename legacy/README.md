# Legacy Archive

Este diretório guarda código frio preservado por referência técnica e recuperação histórica.

## Regra de uso

- não faz parte da superfície operacional suportada
- não entra no fluxo canônico descrito em `README.md`, `docs/00_Governance/` ou `research/README.md`
- pode servir como referência para decisões passadas, compatibilidade e resgate de lógica antiga

## Estrutura

- `legacy/gmail-flow/`: antigo fluxo Gmail/e-mail/coleta, incluindo `upwork_jobs.db`
- `legacy/experiments/`: scripts soltos, simulações e testes descartados da raiz

## Status

Cold archive. Preservado, mas não mantido como runtime principal.
